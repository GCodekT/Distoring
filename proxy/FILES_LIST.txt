💡 ЗАЧЕМ НУЖЕН ПРОКСИ?

SIM800L не работает с HTTPS! Решение:
  Arduino → HTTP → Прокси → HTTPS → Основной сайт

Преимущества:
  ✓ Стабильная работа SIM800L
  ✓ Основной сайт защищен SSL
  ✓ Очередь данных при сбоях
  ✓ Автоматическая повторная отправка
