❓ Часто задаваемые вопросы

Как добавить свой датчик в систему? ▼
1. Соберите устройство на базе Arduino + SIM800L + MPU6050
2. Прошейте код из репозитория, добавив уникальный ID датчика
3. Измените URL в коде на адрес вашего сервера
4. После первой успешной отправки данных датчик автоматически появится на карте
5. Зарегистрируйтесь на сайте и добавьте датчик в свой аккаунт
Как получить точные координаты датчика? ▼
По умолчанию SIM800L определяет координаты базовой станции GSM (точность 200-2000м). Для точных координат:
• Добавьте GPS модуль (например, NEO-6M)
• Или зарегистрируйтесь и установите точные координаты вручную в личном кабинете
Какой формат данных отправляет датчик? ▼
Формат: V:12.50,C:85,R:0.50,P:-1.20,S:OK

Где:
• V - Напряжение батареи (вольты)
• C - Заряд батареи (проценты)
• R - Крен/Roll (градусы)
• P - Тангаж/Pitch (градусы)
• S - Статус (OK, LOW_BATTERY, и т.д.)

URL: api_log.php?id=YOUR_ID&a=V:12.5,C:85,R:0.5,P:-1.2,S:OK
Как часто датчик отправляет данные? ▼
В текущей версии: каждые 2 минуты (можно настроить в коде Arduino).
Рекомендуемые интервалы:
• Для мониторинга движения: 30-60 секунд
• Для статичных объектов: 5-10 минут
• Для экономии батареи: 30-60 минут
Можно ли использовать другие GSM модули? ▼
Да! Любые модули с AT-командами:
• SIM800L (рекомендуется)
• SIM900/SIM900A
• A6/A7
• ESP8266/ESP32 с GSM shield

Нужно адаптировать AT-команды в коде Arduino под ваш модуль.
Сколько стоит поддержка сервера? ▼
Система полностью бесплатная и открытая (Open Source).
Вам нужен только хостинг с PHP и MySQL:
• Shared хостинг: от $3-5/месяц
• VPS: от $5-10/месяц
• Можно разместить на домашнем сервере бесплатно
Как защищена передача данных? ▼
Рекомендации по безопасности:
• Используйте HTTPS (SSL-сертификат)
• Добавьте аутентификацию по API ключу
• Ограничьте доступ к API по IP
• Используйте уникальные ID для каждого датчика
• Регулярно обновляйте систему
