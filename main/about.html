О проекте

IoT Sensor Monitoring - это открытая платформа для мониторинга датчиков на базе Arduino и GSM модулей в реальном времени.

🎯 Цель проекта

Создать простую и доступную систему для отслеживания IoT устройств с минимальными требованиями к оборудованию. Любой человек с базовыми знаниями Arduino может создать свой датчик и подключить его к системе.

✨ Возможности

🗺️
Интерактивная карта

Визуализация всех датчиков на карте мира в реальном времени

📊
Графики и статистика

Детальный анализ данных с интерактивными графиками

🔐
Система аккаунтов

Управление своими датчиками через личный кабинет

📡
GSM модули

Поддержка SIM800L и определение координат по вышкам

🌐
REST API

Полноценный API для интеграции с другими сервисами

🚀
Open Source

Полностью открытый исходный код для модификации

🛠 Технологии

Backend: PHP 7.4+, MySQL, REST API

Frontend: HTML5, CSS3, JavaScript (ES6+), Leaflet.js, Chart.js

Hardware: Arduino (Nano/Uno), MPU6050, SIM800L GSM модуль

📖 Как начать?

1. Соберите датчик на базе Arduino + SIM800L + MPU6050
2. Прошейте код с уникальным идентификатором
3. Датчик автоматически появится на карте после первой отправки данных
4. Зарегистрируйтесь и добавьте датчик в свой аккаунт

💡 Применение

• Мониторинг наклона объектов (здания, мосты, столбы)
• Охранные системы с датчиками движения
• Мониторинг транспорта и логистики
• Экологический мониторинг
• Умный дом и автоматизация
• Учебные проекты и исследования
